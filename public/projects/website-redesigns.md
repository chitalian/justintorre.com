# 17 versions of this website

In July 2026 I decided my old homepage (a starfield, a fake terminal, a chatbot) was dated. Instead of picking one new design, I built 17 complete homepage concepts in one sitting, then chose the plainest one: v12, black text on white, which became the live site.

Flip through all 17 at https://justintorre.com/overview

## The versions

- v1 Terminal brutalist — tmux session, phosphor green, blinking cursors → https://justintorre.com/redesign/v1
- v2 Swiss editorial — white specimen sheet, huge type, one red accent → https://justintorre.com/redesign/v2
- v3 Bento grid — dark rounded cards, stat tiles, glow on hover → https://justintorre.com/redesign/v3
- v4 Builder changelog — life as a CHANGELOG.md → https://justintorre.com/redesign/v4
- v5 One continuous line — Tracer homage, a single stroke threads the page → https://justintorre.com/redesign/v5
- v6 justinOS — fake desktop, windows, dock, menu bar clock → https://justintorre.com/redesign/v6
- v7 The Torre Times — broadsheet front page on cream paper → https://justintorre.com/redesign/v7
- v8 Quiet prose — one narrow column, typography only → https://justintorre.com/redesign/v8
- v9 Aurora hero — drifting gradients, glass cards, stats marquee → https://justintorre.com/redesign/v9
- v10 Mission control — ops dashboard, KPI cards, status pills → https://justintorre.com/redesign/v10
- v11 Retro-future — synthwave sunset, perspective grid, high scores → https://justintorre.com/redesign/v11
- v12 Black and white — no frills, sharp 1px rules; the winner → https://justintorre.com/redesign/v12
- v13 Classic Macintosh — System 7 windows, pinstripe title bars → https://justintorre.com/redesign/v13
- v14 Neon receipt — v12's skeleton in v11's skin → https://justintorre.com/redesign/v14
- v15 Field guide — dark blueprint, wireframe spheres, step cards → https://justintorre.com/redesign/v15
- v16 Analog mission control — TVA dossier, telemetry meters → https://justintorre.com/redesign/v16
- v17 Modern steampunk — brass gears, colored filigree → https://justintorre.com/redesign/v17

## The lesson

After building synthwave grids and fake operating systems, the design that won was black text on white with good spacing. There is probably something to that.

The whole thing is in the repo: https://github.com/chitalian/justintorre.com
